异步加载 js 脚本

2019-07-24 #js基础

script 中加入 async(html5) 或者 defer(html4) 属性，脚本会异步加载。

区别

defer

  1. defer 要等到页面在内存中正常渲染结束，在 window.onload 之前执行
  2. 如果有多个 defer，会按照在页面的出现顺序加载

async

  1. async 一旦下载完，渲染引擎就会中断渲染，执行这个脚本，再继续渲染
  2. 多个 async 不能保证顺序
